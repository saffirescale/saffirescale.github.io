Github-Actions-Sonarqube-Quality-Notification

To demonstrate CI/CD Pipeline to automate sonarqube quality check on code push in the github repository using Github Actions & Sonarqube cloud. Also send notification email to user with sonar analysis link.

Workflow

  1. Step 1: Github Actions trigger on code push to github.
  2. Step 2: Execution steps shown for sonarqube and notification event.
  3. Step 3: Sonarqube cloud link of project quality analysis.
